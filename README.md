## 电路参数 | parameters

|             参数 parameter             | 值 value | 单位 unit |
|:------------------------------------:|:-------:|:-------:|
|              输入电压 (Vin)              |  10-15  |    V    |
|             输出电压 (Vout)              | 19 & 5  |    V    |
|  5V输出电流限制 (5V output current limit)  |    3    |    A    |
| 19V输出电流限制 (19V output current limit) |    3    |    A    |
