# minipc-power-board

给minipc供电的12V输入供电板 | 12V input power supply board for minipc

## 电路参数 | parameters

|        参数 parameter         | 值 value | 单位 unit |
|:---------------------------:|:-------:|:-------:|
|         输入电压 (Vin)          |  10-15  |    V    |
|         输出电压 (Vout)         | 19 & 5  |    V    |
| 5V输出电流限制 (5V current limit) |    3    |    A    |
|          19V输出电流限制          |    3    |    A    |
